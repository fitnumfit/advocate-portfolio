# Advocate Iqra Imran — Legal Services Portfolio

A Next.js portfolio website for **Advocate Iqra Imran**, focused on legal services with emphasis on **traffic challan** and Lok Adalat settlement.

## Features

- **Running marquee** — “SPECIAL LOK ADALAT FOR SETTLEMENT OF TRAFFIC CHALLANS” with contact number
- **Legal services** — Criminal, family cases, court marriage, divorce, traffic challan, cheque bounce, contract, agreement, rent agreement
- **Traffic challan** highlighted as primary service
- Responsive layout and professional legal theme

## Update phone number

Edit **`src/config.ts`** and set your contact number:

```ts
export const PHONE_NUMBER = "+91 XXXXX XXXXX";
export const PHONE_LINK = "tel:+91XXXXXXXXXX";
```

The marquee and contact section use this config.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```bash
npm run build
npm start
```

## Tech

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
